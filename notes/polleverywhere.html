CTL: Poll Everywhere

The Poll Everywhere CTL class was a simple overview and introduction to Poll Everywhere. The class demonstrated the following ideas: how faculty and students see and use Poll Everywhere, how to setup Poll Everywhere for faculty and students, how to create poll questions, how to import a canvas course, how to integrate Poll Everywhere into slides and how to download Poll Everywhere extensions. One the more interesting things I took away from the talk is the fact that you can actually use LaTex within Poll Everywhere. While this might seem like quite a simplistic feature, it actually makes writing of mathematical equations much easier. One of the down sides to Poll Everywhere when being used for grading is that only 2 types of questions, multiple choice and clickable images, can be transferred to canvasâ€™s grade book. The rest of the question types must sadly be manually input into the canvas grade book. Besides these two points I found the overall class useful and received a reference sheet to help later down the line when I begin to integrate Poll Everywhere into my own classes.
